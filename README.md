# cart-react

> 🛒 Simple Cart using React.

## Install

Install dependencies.
```
$ npm i
```
Install dependencies for `API` folder.
```
$ cd API && npm i
```

## Usage

One line to run.
```
$ npm start
```
For API folder:
```
$ cd API && npm start
```
